# TikTalk

Социальная сеть для разработчиков. Написана на Angular 18.

[English version](README.md)

## Технологии

- **Angular 18** — фреймворк
- **TypeScript** — язык
- **SCSS** — стили (методология BEM)
- **RxJS** — реактивное программирование
- **Angular Signals** — управление состоянием
- **ngx-cookie-service** — хранение токенов

## Функциональность

- Авторизация с JWT токенами (access + refresh)
- Страница профиля пользователя со списком подписчиков
- Настройки профиля (имя, описание, навыки, загрузка аватара)
- Поиск и фильтрация пользователей
- Навигация в сайдбаре с подсветкой активного роута
- Загрузка аватара через drag & drop
- Textarea с автоматическим изменением высоты
- Переключение видимости пароля

## Запуск

### Требования

- Node.js 18+
- npm

### Установка зависимостей

```bash
npm install
```

### Запуск dev-сервера

```bash
ng serve
```

Открыть `http://localhost:4200` в браузере.

## Регистрация

Для регистрации необходимо использовать Telegram бот **@icherniakov_info_bot**:

1. Открыть Telegram и найти `@icherniakov_info_bot`
2. Отправить `/start`
3. Нажать синюю кнопку в ответе бота
4. Бот пришлёт **логин** и **пароль** для входа на сайт

## Структура проекта

```
src/
├── app/
│   ├── auth/               — авторизация (сервис, интерсептор, гард)
│   ├── common-ui/          — общие компоненты (сайдбар, лейаут, хедер профиля и др.)
│   ├── data/               — сервисы и интерфейсы
│   ├── helpers/            — пайпы и директивы
│   └── pages/              — страницы
│       ├── login-page/
│       ├── profile-page/
│       ├── search-page/
│       └── settings-page/
└── public/
    └── assets/             — шрифты, изображения, SVG иконки
```
